Manual de Usuario

Este documento proporciona una guía práctica para el uso del formualrio de Cierre Contable.

1. Cierre Contable

El sistema permite realizar cierres contables de dos tipos:

  • Cierre Mensual
  • Cierre Anual

El contador puede seleccionar el tipo de cierre desde el formulario principal.

2. Cargar Cuentas

El usuario debe seleccionar los parámetros:

  • Fecha desde
  • Fecha hasta
  • Tipo de actualización (En línea o Batch)

Tras seleccionar los parámetros, el sistema cargará todas las cuentas dentro del rango elegido, mostrando:

  • Total del Debe
  • Total del Haber
  • Saldo total del cierre

3. Validaciones del Sistema

Para garantizar la integridad de los datos, el sistema aplica diversas validaciones:

  • Un cierre mensual solo puede realizarse una vez por mes.
  • Un cierre anual solo se puede ejecutar una vez por año, aun si no hubo movimientos.
  • El sistema valida automáticamente si existe un cierre previo.

Si no existe un cierre previo, el sistema permitirá guardar y mostrará un mensaje de confirmación.

4. Reporte de Cierre Contable

El sistema permite generar un reporte con todos los movimientos relacionados al cierre contable, facilitando el análisis y auditoría.

Documento generado automáticamente como material de apoyo al usuario.
